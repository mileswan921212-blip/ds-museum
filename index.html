Arrays

Arrays are a basic data structure that stores elements in a continuous memory block.

10
22
35
47
53
66

Example Code (C++)

int arr[6] = {10, 22, 35, 47, 53, 66};

cout << "Array elements: ";
for (int i = 0; i < 6; i++) {
    cout << arr[i] << " ";
}

Stacks

A stack is a Linear Data Structure following LIFO (Last In, First Out).

5
12
18
33

Example Code (C++)

stack<int> st;

st.push(5);
st.push(12);
st.push(18);
st.push(33);

cout << "Top element: " << st.top();
st.pop();
